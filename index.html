Proyectos

Proyecto 1

Este proyecto ofrece una API que permite acceder a información detallada sobre los personajes, episodios y ubicaciones del mundo de Rick and Morty. Los usuarios pueden obtener datos sobre las características de los personajes, como sus nombres, especies, orígenes, y estados de salud, así como los episodios en los que aparecen y las dimensiones a las que viajan. Esta API es ideal para crear aplicaciones y herramientas que exploran el extenso y cómico universo de la serie.

Proyecto 2

Este proyecto proporciona una API que centraliza información sobre el universo Pokémon, ofreciendo datos detallados sobre los diferentes Pokémon, sus estadísticas, habilidades, movimientos, evoluciones y tipos. Además, permite acceder a información sobre los entrenadores, objetos, y las ubicaciones dentro del mundo Pokémon, facilitando a los desarrolladores la integración de estos datos en sus aplicaciones, juegos y proyectos educativos.

HTML5

"HTML5 es el estándar actual para la estructura y el diseño de páginas web. Ofrece nuevas funcionalidades que mejoran la accesibilidad, la multimedia y la interactividad. Es fundamental para el desarrollo de aplicaciones web modernas y se integra perfectamente con CSS3 y JavaScript."

CSS3

"CSS3 es el lenguaje utilizado para darle estilo y diseño a las páginas web. Permite crear efectos visuales avanzados como animaciones, transiciones y nuevos controles de diseño, como las 'media queries', que facilitan la creación de interfaces adaptables y responsivas."

JavaScript

"JavaScript es el lenguaje de programación que permite añadir interactividad a las páginas web. Es esencial para manipular el DOM (Document Object Model), manejar eventos de usuario y crear aplicaciones web dinámicas y reactivas."

PHP

"PHP es un lenguaje de programación del lado del servidor ampliamente utilizado para desarrollar aplicaciones web dinámicas. Es perfecto para la creación de sitios web interactivos que requieren bases de datos, formularios y autenticación de usuarios."

GitHub

"GitHub es una plataforma de control de versiones que permite a los desarrolladores gestionar sus proyectos de software de manera eficiente. Es fundamental para el trabajo colaborativo, la documentación y el seguimiento de cambios en el código fuente a través de 'commits' y 'branches'."
